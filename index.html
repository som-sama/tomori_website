Usage

To have a conversation with Tomori, simply start a message without any special prefix, and Tomori will respond based on the conversation history.

To set a reminder, use the command "-remind" followed by the duration in hours and the reminder message.

-remind 2 Buy groceries

Tomori will remind you of the specified message after the given duration.

To check the time left for the reminder, use the command "-left".

-left

Tomori will reply with the remaining time for the set reminder.
